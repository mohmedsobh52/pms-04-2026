

# تحديث شامل للواجهة والخلفية - داشبورد احترافي لادارة المشاريع

## الهدف

ترقية التصميم الحالي الى واجهة احترافية حديثة تتناسب مع برامج ادارة المشاريع الانشائية الكبرى، مع خلفية انيقة وداشبورد متقدم.

---

## التحديثات المطلوبة

### 1. خلفية احترافية جديدة

**ملف**: `src/components/BackgroundImage.tsx`

التصميم الجديد:
- **تدرج لوني عميق** بزوايا متعددة (من الازرق الداكن للرمادي الفولاذي) بدلا من slate مسطح
- **شبكة هندسية دقيقة** (Blueprint Grid) بنمط المخططات الانشائية - خطوط رفيعة جدا بشكل احترافي
- **توهج ديناميكي** واحد يتغير حسب المرحلة النشطة لكن بشكل اكثر نعومة
- **تأثير Vignette** خفيف على الحواف لاعطاء عمق
- حذف الشبكة القديمة والكرة العائمة

### 2. داشبورد احترافي متقدم

**ملف**: `src/pages/HomePage.tsx`

التصميم الجديد يشمل:

```text
+----------------------------------------------------------+
| Header (موحد)                                             |
+----------------------------------------------------------+
| Welcome Banner (تدرج لوني + اسم + تاريخ + زر سريع)      |
+----------------------------------------------------------+
| 4 KPI Cards متقدمة (مع Mini Sparkline + نسبة تغير)       |
| المشاريع | القيمة | العقود | المخاطر                      |
+----------------------------------------------------------+
| Recent Projects (مع Progress Bar)  | Quick Access (محسن)  |
| - شريط تقدم لكل مشروع             | - ايقونات اكبر       |
| - حالة المشروع ملونة              | - وصف اوضح          |
+----------------------------------------------------------+
| Project Lifecycle (محسن)                                  |
+----------------------------------------------------------+
| Phase Actions (محسن)                                      |
+----------------------------------------------------------+
```

التحسينات:
- **Welcome Banner**: تدرج لوني جذاب مع رسالة ترحيب ورقم المشاريع النشطة وزر "مشروع جديد" سريع
- **KPI Cards محسنة**: اضافة مؤشر اتجاه (سهم اعلى/اسفل)، خلفية متدرجة خفيفة، حجم خط اكبر للرقم الرئيسي
- **Recent Projects محسنة**: اضافة شريط تقدم صغير لكل مشروع، badge حالة ملون (نشط/مكتمل/معلق)، تصميم بطاقات اوضح
- **Quick Access محسن**: ايقونات اكبر مع خلفيات ملونة، تنظيم افضل

## التفاصيل التقنية

### الخلفية الجديدة (Blueprint Style)

- تدرج: `from-slate-950 via-blue-950/50 to-slate-950` - لمسة زرقاء عميقة
- شبكة Blueprint: خطوط بنمط هندسي `opacity-[0.04]` مع `backgroundSize: 40px`
- Aurora blob: واحد فقط، اكبر حجما `w-[600px] h-[500px]`، شفافية `0.08`
- Vignette: `radial-gradient(ellipse at center, transparent 50%, background 100%)`

### KPI Cards المحسنة

كل بطاقة تشمل:
- ايقونة في دائرة ملونة (gradient)
- رقم كبير (text-3xl font-bold)
- وصف صغير (text-xs text-muted-foreground)
- خط حدود علوي ملون (border-t-2) بلون مميز لكل بطاقة

### Welcome Banner

- خلفية: `bg-gradient-to-r from-primary/10 via-accent/5 to-transparent`
- حدود: `border border-primary/20`
- محتوى: تحية + تاريخ + عدد المشاريع النشطة + زر CTA
